# *SUMMARY*

  천재 IT교육센터 자바 풀스택 과정 교육시간 내 작성한 프로그램 중, 그 날 학습한 내용 가운데 핵심적인 주제를 다루었으나 <br>
  개별 레포지토리에 정리하기에는 로컬 프로젝트 폴더의 규모가 작은 것들을 따로 모아 정리하였다.
<br><br>
